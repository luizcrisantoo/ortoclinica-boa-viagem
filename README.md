# Ortoclínica Boa Viagem — Site Institucional

Site one-page da **Ortoclínica Boa Viagem**, clínica de traumato-ortopedia e fisioterapia em Boa Viagem, Recife-PE. Construído em HTML, CSS e JavaScript puros (sem frameworks), seguindo o padrão **GSD PRO** de estrutura, segurança e performance.

**Conversão via WhatsApp** · **Público-alvo 50+** · **100% estático**
